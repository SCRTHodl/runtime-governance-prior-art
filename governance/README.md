# Governance Specifications

This directory contains normative, non implementational specifications that operationalize the principles described in the reference paper.

These documents define invariants and conformance requirements.  
They do not disclose implementation details.

- Authority Admission Layer: `./AUTHORITY_ADMISSION.md`
- Governance Continuity: `./GOVERNANCE_CONTINUITY.md`

## Reference

## Notice

These specifications are normative and non implementational.
They describe constraints and required behavior, not mechanisms, algorithms, key material, configuration, or system designs.
